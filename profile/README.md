### Hey, we're Elementeer

We build the agent-native growth layer for Elementor — an open platform that connects AI agents directly to WordPress and Elementor through structured APIs, MCP servers, governance systems, and workflow orchestration.

WordPress + Elementor powers over 15 million websites, but agent-driven creation and management remains fragmented. Elementeer bridges this gap with a secure, capability-scoped REST API and a growing ecosystem of tools, wizards, and bridges.

---

### What we do

Elementeer gives AI agents real operational access to Elementor sites:

- **Elementeer Plugin** — WordPress plugin exposing the full Elementor template library over a typed REST API (`/wp-json/elementeer/v1/`). No 401 surprises, no empty responses. GPL-2.0-or-later.

- **Elementeer MCP** — Node.js MCP server with 111+ tools. Connects Claude, OpenCode, Cursor, and other MCP clients directly to your Elementor library. Apache-2.0.

- **Elementeer Bridge** — Headless conversion engine that transforms any website (URL, HTML, or image) into Elementor-compatible JSON templates via AI. Apache-2.0.

- **Elementeer Ops** — Docker-based WordPress testing environment with Elementor, Voxel, and premium bundle integration. Apache-2.0.

### Get Started

```bash
# Install the MCP server
npm install -g @elementeer/mcp

# Or via Capacium Exchange
cap install elementeer-mcp

# Initialize your first connection
elementeer-mcp init
```

1. Install the [Elementeer WordPress Plugin](https://github.com/elementeer/elementeer/releases)
2. Generate an API key in Settings → Elementeer
3. Configure your MCP client with the key
